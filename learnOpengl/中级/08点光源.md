# 实现
目标：
1. 加入新的材质（纯白色）表示灯泡光源（之前只有phongMaterial）
2. 加入`whiteShader`（之前的`phong`模型肯定不适合搞这个），用于白色材质渲染
3. 在`Render`函数中加入白色材质的解析与渲染（之前的`switch`分支中只有`phong`的）
4. 加入`PointLight`类

### 目标一
在`material.h`中，添加一个新的枚举`WhiteMaterial`
```
enum class MaterialType {
	PhongMaterial,
	WhiteMaterial
};
};
```
