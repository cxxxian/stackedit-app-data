# 理论设计
前向渲染是：**每渲染一个物体 → 计算完整光照 → 输出颜色**
延迟渲染是：

1.  第一步（GBuffer Pass）：渲染所有物体，把几何信息写入多个缓冲（GBuffer）：
    位置 / 法线 / 漫反射颜色 / 镜面反射参数 等
        
2.  第二步（Lighting Pass）：遍历光源，**在屏幕空间**中进行光照计算，生成最终颜色
