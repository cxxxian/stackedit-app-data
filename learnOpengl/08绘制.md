### 绘制三角形
glDrawArrays(GL_TRIANGLES, 0, 3);
此处的0 ，3，分别意为，**从vertices[0]开始，取三个顶点进行绘制**
```
void render(){
    //执行opengl画布清理操作
    glClear(GL_COLOR_BUFFER_BIT);

    //1 绑定当前的program
    glUseProgram(program);
    //2 绑定当前的vao
    glBindVertexArray(vao);
    //3 发出绘制指令
    glDrawArrays(GL_TRIANGLES, 0, 3);
}
```
