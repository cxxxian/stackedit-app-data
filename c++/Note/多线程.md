## C++ 多线程的基本用法：`std::thread`

从 C++11 开始，我们可以用 `std::thread` 创建新线程。

### 示例：创建一个线程打印数字

```cpp
#include <iostream>
#include <thread>  
void printNumbers() {     
	for (int i = 1; i <= 5; ++i) {         
		std::cout << "子线程输出: " << i << std::endl;     
	} 
}  
int main() {     
	std::thread t(printNumbers);// 启动新线程执行 printNumbers     		t.join();                     // 等待线程执行完成     std::cout << "主线程结束" << std::endl;     return 0; }
```

### 运行结果：

```makefile
子线程输出: 1 
子线程输出: 2
... 
主线程结束
```
