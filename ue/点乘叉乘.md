## 点乘叉乘应用
**ue5**使用的是左手坐标系
### 点乘
即A * B = |A| * |B| * cos（Theta）；
将A, B单位化即可得到
A * B = cos（Theta）；
先算出点乘（cos（Theta））的结果，再用ACos反余弦得到真正的Theta值，最后用RadiansToDegrees(Theta)即可得到Theta的度数。
```
const double CosTheta = FVector::DotProduct(Forward, ToHit);
double Theta = FMath::Acos(CosTheta);
Theta = FMath::RadiansToDegrees(Theta);
```
